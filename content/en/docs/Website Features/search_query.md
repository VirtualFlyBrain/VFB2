VFB integrates data curated from the literature with image data from many sources. Searches and queries can be used to explore this data.

 <img src="/images/search_query/search_query_location.png" max-width="50%" alt="Location of the VFB search and query tools.">

## The Search Tool
The search system allows you to search for neurons and neuroanatomical structures using almost any name found in the literature. Positive and negative filters can be applied from the drop-down to restrict the results.

<img src="/images/search_query/search_tool.png" max-width="50%" alt="The VFB search tool.">

## The Query Tool
The query system can identify neurons innervating any specified neuropil or fasciculating with any specified tract. It also allows queries for genes, transgenes and phenotypes expressed in any brain region or neuron.

<img src="/images/search_query/query_tool.png" max-width="50%" alt="The VFB query tool.">

## Queries from Term Info

Some Term Info panes have pre-defined queries relevant to that term. For example, the Term Info pane for a brain region may have queries for neurons innervating that region and drivers expressing in that region.

<img src="/images/search_query/terminfo_queries.png" max-width="50%" alt="Queries in the Term Info pane.">

## Query Results
Query results give information for each result, including [thumbnail images](thumbnails) where available. Results can be filtered by typing in the top bar and sorted by clicking on column titles.

<img src="/images/search_query/query_results.png" max-width="50%" alt="Results from an example query on VFB.">
